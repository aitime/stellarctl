stellarctl
==========

The long missing stellar command line utility!

## Features

### functional

* create accounts
* send lumens and other assets
* manage trust
* issue assets
* generate keys
* set account options
  * inflation address
  * thresholds
  * signers
  * ...
* inspect ledger for account details
  * transactions
  * operations
  * effects
* inspect transaction details
  * operations
  * effects
* create test accounts using friendbot

### non-functional

* single go binary
* uses official go SDK as far as possible
* simple command line syntax
